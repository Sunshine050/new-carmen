กด Approved PR ไม่ได้  
ตัวอย่าง PR24020004 กดเข้ามาแล้วไม่พบปุ่มปุ่ม Approved/Reject/Send Back ให้กด  
![](_images/7.กด Approved PR ไม่ได้/img-001.png)

# วิธีเพิ่มAddinCarmenกลับคืน

Tab Carmen Add-In หายไป จะเพิ่มกลับมาอย่างไร  
วิธีแก้ไขเพิ่ม add in กลับมา  

![img-001.png](/carmen_cloud/faq/_images/restore-carmen-addin-excel/img-001.png)

1.Click “File”

![img-002.png](/carmen_cloud/faq/_images/restore-carmen-addin-excel/img-002.png)

Click “option”

![img-003.png](/carmen_cloud/faq/_images/restore-carmen-addin-excel/img-003.png)

2.Click “Add-ins”  

3.Cick GO  

![img-004.png](/carmen_cloud/faq/_images/restore-carmen-addin-excel/img-004.png)

4. Check box at “Carmen AddIn”

5.Click “OK” to complete the process.  

![img-005.png](/carmen_cloud/faq/_images/restore-carmen-addin-excel/img-005.png)

Excel จะแสดง Carmen Add-in ขึ้นมาตาม สามารถดำเนินการใช้งานได้ตามปกติ

![img-006.png](/carmen_cloud/faq/_images/restore-carmen-addin-excel/img-006.png)

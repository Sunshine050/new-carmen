# กด Approved PR ไม่ได้

กด Approved PR ไม่ได้ 

ตัวอย่าง PR24020004 กดเข้ามาแล้วไม่พบปุ่มปุ่ม Approved/Reject/Send Back ให้กด  
![img-001.png](/carmen_cloud/faq/_images/approved-pr-button-not-visible/img-001.png)

สาเหตุเกิดจากการเปิดเอกสารเมื่ออยู่ในหมวด View All ทำให้ไม่สามารถแก้ไขได้

![img-002.png](/carmen_cloud/faq/_images/approved-pr-button-not-visible/img-002.png)

ไปที่หัวข้อ View  หรือตาม View Step เอกสารPR ของลูกค้า  

![img-003.png](/carmen_cloud/faq/_images/approved-pr-button-not-visible/img-003.png)

ทำการคลิกที่PR24020004หรือหมายเลขPR ของลูกค้า

![img-004.png](/carmen_cloud/faq/_images/approved-pr-button-not-visible/img-004.png)

จะพบว่าปุ่ม Approved/Reject/Send Back ปรากฏขึ้นมาแล้วตามรูปภาพ

![img-005.png](/carmen_cloud/faq/_images/approved-pr-button-not-visible/img-005.png)
